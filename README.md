# Regex Course

Exercises and examples from a Regex course on Udemy.
Covers regex fundamentals through advanced features, with examples in JavaScript, Python, Java, and Go.

## Structure

```
regex-course/
├── characters/         # Characters: simple, meta, whitespace, unicode, OR
├── sets/               # Sets: ranges, negation, shorthands
├── dot/                # Dot metacharacter
├── quantifiers/        # Quantifiers: ?, *, +, {n,m}, greedy vs. lazy — includes real-world challenges
├── groups/             # Groups and backreferences
└── runningRegex/       # Running regex in JS, Python, Java, Go, Ruby — flags and methods
```

## Topics covered

- Simple characters, metacharacters, whitespace, and Unicode
- Sets, negated sets, ranges, and shorthands (`\d`, `\w`, `\s`)
- Dot, OR, and common pitfalls
- Quantifiers: `?`, `*`, `+`, `{n,m}`
- Greedy vs. lazy (non-greedy) matching
- Capturing groups and backreferences
- Flags: `g`, `i`, `m`
- Real-world challenges: CPF, email, and phone number validation
- Regex in JavaScript, Python, Java, Go, and Ruby
